Installation

Installing Theme Video

You should install the latest version of WordPress in your system. This will ensure you're up to date, and will allow the theme to use all the new features of WP 3.7+ Versions.

Please don't upload whole lms-1.2.zip archive.

 

You have a two ways to install a theme.

 

1 - First way

Please take a look a structure of the lms-1.2.zip archive you got after purchasing. Upload a theme folder only on the ../wp-content/themes/ directory via any FTP manager. Then follow to Admin Panel > Appearance > Themes and activate a new theme.

 

 

2 - Second way

Find a theme folder (see image above) and zip that. Then follow to Admin Panel > Appearance > Themes > Install Themes > Upload and upload an archive you've created.

 

 

Installing the dummy data

There are two ways to install the dummy data.

1. One click install - We have given option in BPanel.

2. Install the dummy data which is included in the directory "lms/framework/wordpress-importer/dummy/lms-dummy.xml". This will ensure the posts/pages get laid out properly with the correct page templates. Go to Tools > Import > Wordpress. Select the XML file and upload it. Be sure to click "download and import all attachments"

Setting up Mega Menu

The theme offers 2 menu locations: the Header Menu location beside your logo and a very small Top Menu location at the top of your page, right above your logo. Once you have created a menu you then have to select where to display it. Please keep in mind that the small menu location above your logo doesnt support dropdown menus or mega menus.

Our theme supports mega menu. You can enable it by checking the Use as Mega Menu checkbox as shown below.

You have an option to choose the number of columns for the mega menu. Also You can disable the link or text for the column title.

Setting up Home Page

Create new page and named it as Home Page or anything you want.

Go to 'Settings > Reading' and choose a static page as front page.

Folder Structure

  • style.css - main style.css
  • responsive.css - makes the template as responsive
  • The skins folder - contains the css for predefined skins that comes with the template.
  • The css folder - contains the css for all major browsers, prettyphoto & for the font awesome. - ( Please ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours. )
  • The images folder - contains all the images that are used in the stylesheets for the layout purpose & it has subfolder named post-images , which contains images used in post, pages and other areas
  • The js folder - contains javascript files & plugins ( Which includes)
    1. Easing (easing.js)
    2. Modernizr (modernizr-2.6.2.min.js)
    3. Mean Menu (jquery.meanmenu.js)
    4. Isotope plugin (isotope.js)
    5. Nice Scroll (jquery.nicescroll.min.js)
    6. CarouFredSel plugin (jquery.carouFredSel-6.2.0-packed.js)
    7. PrettyPhoto (jquery.prettyPhoto.js)
    8. Tabs (jquery.tabs.min.js)
    9. bx Slider plugin (jquery.bxslider.min.js)
    10. video ratio(jquery.fitvids.js)
    11. Smart Resize (jquery.smartresize.js)
    12. And custom js like (custom.js)
